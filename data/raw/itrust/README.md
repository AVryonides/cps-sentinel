# iTrust SWaT data

The SWaT dataset must be requested directly from iTrust. Its terms prohibit sharing the
dataset with others, including through a public or private Git repository.

Place authorized local files in this directory; all contents except this README are ignored.
Never attach SWaT data to a release or commit it with Git LFS.

Official access and terms:
https://itrust.sutd.edu.sg/itrust-labs_datasets/dataset_info/
